☁️ Day 1 – Introduction to Cloud & AWS Basics
1️⃣ Understand AWS Pricing Models
💰 Major AWS Pricing Models
Pricing Model	What It Means	Best For
Pay-As-You-Go	Pay only for what you use	Beginners, learners, unpredictable workloads
Reserved Instances (RI)	Commit for 1–3 years for discount	Stable enterprise workloads
Spot Instances	Use unused AWS capacity at big discount	Batch jobs, testing
Savings Plans	Flexible commitment-based discount	Growing startups




2️⃣ Compare Cloud Deployment Models
☁️ On-Premises
Infrastructure located inside organization

High control

High maintenance cost

Used by: Banks, Government, defense




🌍 Cloud
Hosted by cloud providers

Scalable

Low upfront cost

Used by: Startups, SaaS companies, learners




3️⃣ Service Models – IaaS, PaaS, SaaS
🏗️ IaaS (Infrastructure as a Service)

You manage OS, apps. Provider manages hardware.

Examples:

Amazon Web Services Amazon EC2

Amazon S3

Microsoft Azure Virtual Machines

⚙️ PaaS (Platform as a Service)

You deploy code. Provider manages infrastructure.

Examples:

AWS Lambda

AWS Elastic Beanstalk

Google App Engine

📧 SaaS (Software as a Service)

Fully managed software.

Examples:

Amazon WorkMail

Salesforce

Zoom
